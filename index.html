102 ETIQUETAS HTML
¿Que es una Etiqueta HTML?
Una etiqueta HTML es un componente básico del lenguaje HTML utilizado para definir la presentación y estructura del contenido en una página web. Cada etiqueta indica al navegador cómo debe mostrar o interpretar el contenido asociado, como texto, imágenes, enlaces o elementos multimedia. Las etiquetas suelen estar delimitadas por corchetes angulares y pueden tener atributos que añaden información adicional o modifican su comportamiento
